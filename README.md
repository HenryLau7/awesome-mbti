# awesome-bti

一个收录各种 X-BTI / MBTI-like 衍生人格测试、研究演示与社区项目的 curated list。

> 说明：这里的项目横跨娱乐、梗图、研究展示与半严肃工具，**科学有效性**差异很大。默认把它们视为互联网文化项目或产品案例，而不是心理测量学背书。

## 这是什么

`awesome-bti` 关注的是各种“像 MBTI、但又不是传统 MBTI”的衍生项目：

- 爆火的二创人格测试
- 披着研究或实验外衣的互动 demo
- 面向职业、关系、社交、兴趣圈层的变体
- 围绕 X-BTI 生态做的整理、镜像、工具与集合

这个仓库的目标不是争论哪一种最“准”，而是把值得参考、传播或研究的项目放到同一个入口里，方便浏览、对比和继续扩展。

## 如何阅读这个列表

- 按“测试风格 / 使用场景”分类，而不是按技术栈分类。
- 每个条目都尽量保留原始入口，并显式给出 `Tags:`，帮助快速判断它更偏娱乐、研究、社区还是工具。
- 同一个项目只放在一个主分类里；如果它同时跨多个风格，通过标签补充说明。

## 标签说明

- `viral`: 已形成明显传播或社交分享热度
- `parody`: 明显带戏仿、吐槽、梗感
- `research`: 直接关联研究语境、论文、实验或研究展示
- `research-inspired`: 借用了研究展示风格，但不一定对应正式研究结论
- `career`: 更偏职业、工作、生产力语境
- `relationship`: 更偏关系、沟通、社交互动语境
- `fandom`: 更偏圈层、IP、亚文化或兴趣社群语境
- `community`: 典型社区传播、二创或群众参与项目
- `repo`: 有公开代码仓库
- `demo`: 有可直接访问的在线演示
