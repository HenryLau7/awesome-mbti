## Projects

| Project | Description | Links | Tags |
|:--------|:------------|:------|:-----|
| **SBTI** | 近年最具传播性的 X-BTI 代表之一，把 MBTI 式问卷改造成更强互联网梗感、分享感和角色标签感的人格测试。 | [Demo](https://www.sbti.to/en/) / [Repo](https://github.com/UnluckyNinja/SBTI-test) | `viral` `parody` `community` |
| **LXSTI (留学生人格测试)** | 留学生生存图鉴主题的 16 人格测试，以留学生活场景为切入点，匹配不同的留学生存人格类型。 | [Demo](https://lxsti.dev/) | `viral` `community` |
| **research-mbti** | 以 research demo 的表达方式包装 MBTI-type 体验，兼有在线展示和代码仓库，适合作为 research-inspired 项目参考。 | [Demo](https://research-mbti.com/) / [Repo](https://github.com/HenryLau7/research-mbti) | `research` `research-inspired` |
| **ABTI (学术人格测试)** | 面向科研工作者的趣味人格测试，覆盖硕博到 PI 各阶段。30 道科研场景题，5 大模型、15 维度，匹配 26 种科研人格类型。 | [Demo](https://abti.fun/) / [Repo](https://github.com/Traveller2001/ABTI) | `research-inspired` `career` |
| **历史学人格测试** | 基于四大维度、32 道题目，定位你的历史学人格类型，匹配陈寅恪、钱穆、布罗代尔、黄仁宇等 16 位史学宗师。 | [Demo](https://xwxq55hqiyjx.meoo.info/#/) | `fandom` `community` |
| **TSTI (泰勒斯测试)** | Taylor Swift 主题的人格测试，围绕霉霉粉丝圈层设计。原作者：慕容元夜@小红书。 | [Demo](https://tsti.murouane.com/) | `fandom` `community` |
